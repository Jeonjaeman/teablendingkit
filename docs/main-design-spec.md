## 1. 목표

판매가 아니라 **브랜드 경험과 여백**을 우선하는 메인 페이지. 사용자가 들어왔을 때
"무언가를 사야겠다"가 아니라 "잠시 쉬어가고 싶다 / 이 브랜드의 삶이 궁금하다"를
느끼게 한다.

## 2. 핵심 컨셉 — Immersive × Meditative

레퍼런스 [immersive-g.com](https://immersive-g.com/)(Immersive Garden)의 **연출 기법**
(풀스크린 스크롤 스토리텔링, 키네틱 타이포, 3D 오브제, 영상 프레임, 패럴랙스)을
가져오되, **톤은 의뢰자의 명상·자연주의 어스톤**으로 변환한다.

> 다크·역동 WebGL → 아이보리·세이지·클레이 어스톤 + 느린 호흡.
> "인문학적으로 풀어낸다"는 방향에 맞춰, 스크롤을 따라 한 문장씩 드러나는 에세이형 구성.

긴장 해소: 의뢰자는 "과한 움직임 지양"을 명시 → 모션은 느리고 절제된 페이드·시차로만 사용.

## 3. 정보 구조 (6 섹션 + 푸터)

1. **Hero** — 풀스크린 배경 영상 + 철학적 카피
   "차를 만드는 방법보다, 차를 마시는 시간이 더 중요합니다"
2. **브랜드 스토리** — blend since 1975 · 청주 ↔ Chula Vista
3. **취향의 권리** — 핵심 철학 + 회전 3D 찻잔
   "전문가에게는 원칙이 있지만, 소비자에게는 취향의 권리가 있습니다"
4. **티 블렌딩 키트** — 제품 소개 (여기서 처음 상품 등장)
5. **교육 프로그램** — 기관 출강 / 강사 과정 / 기업 워크숍
6. **Journal** — 에세이 · 차 이야기 · 식물 이야기 · 명상
7. **Footer** — "Stay a while." · teablendingkit.kr

## 5. 연출 / 인터랙션

- 스크롤 진입 시 문장 페이드+상승 (IntersectionObserver, 1.3s ease)
- 패럴랙스: 배경 광원 / "since 1975" 시차 이동
- 상단 스크롤 프로그레스 바, Hero "scroll" 낙하 큐
- 3D 찻잔: `<model-viewer>` 자동 회전 + 드래그
- 배경 영상: muted·loop·playsinline (자동재생 차단 시 poster 폴백)

## 7. 카페24 결합 전략 — 독립 랜딩 + 연결

- **메인(이 몰입형 페이지)**: 커스텀 호스팅 + `teablendingkit.kr` 도메인. 연출 자유도 최대.
- **커머스(장바구니·결제·주문·CS)**: 카페24에 위임. "구매/장바구니" CTA만 카페24 상품/몰로 연결.
- 장점: 브랜드 경험 우선 + 결제 안정성. 카페24 스킨 제약에 갇히지 않음.
- B2B·입찰 문의는 별도 폼(게시판/구글폼)으로.
